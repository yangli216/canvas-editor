# 模板可视化编辑深度优化方案

更新日期：2026-05-21

## 设计定位

模板可视化编辑器不是普通表单配置页，而是面向医院病历文书的低代码模板设计工作台。它需要同时支持结构设计、版式设计、字段绑定、规则配置和实时预览。

## 借鉴对象

- Figma：顶部上下文信息、选区反馈、画布网格和组件化设计心智。
- Webflow：左侧组件库、中心画布、右侧属性面板的专业编辑器布局。
- Retool / Appsmith：低代码组件库、属性配置面板、即时反馈和状态提示。
- Ant Design Form Builder：表单字段物料、栅格布局、属性面板和校验规则配置。

## 体验问题审视

1. 左侧物料库缺少搜索、分组和说明，医疗片段较多时查找成本高。
2. 顶部只展示基础输入和按钮，缺少模板复杂度、选区、配置问题和保存状态反馈。
3. 中心画布缺少专业设计工具的画布语境，块卡片的层级、编号、摘要和类型识别不足。
4. 空画布提示较弱，对新用户没有给出“先搭结构再放字段”的编辑路径。
5. 属性面板可用但视觉层级偏平，和工作台整体质感不够统一。

## 优化目标

1. 视觉：编辑器从“功能面板拼接”提升为“专业设计工作台”。
2. 交互：物料可搜索、可分组；画布块有编号、摘要、类型色条；顶部随编辑变化实时反馈。
3. 信息：持续展示结构块、字段、规则、页眉、页脚、配置问题等关键指标。
4. 效率：通过医疗片段、清晰空状态和选区提示降低模板搭建成本。
5. 扩展：保留三栏设计器结构，为后续拖拽、缩放、缩略图、字段绑定中心继续演进。

## 已落地改造

### 顶部工作台

- 标题升级为“可视化模板设计器”。
- 增加设计器说明：结构、版式、字段和规则一体化编排。
- 增加已保存 / 未保存状态提示。
- 增加选区提示：正文/页眉/页脚中的块或字段。
- 统一预览、保存、关闭等按钮的视觉风格，形成同一套主按钮 / 次按钮 / 轻按钮体系。

### 设计与预览布局

- 当工作台宽度足够时，设计画布与预览画布左右分栏同时展示。
- 当工作台宽度不足时，自动回退为“设计 / 预览” tab 模式，避免横向压缩影响编辑。
- 分栏模式下，编辑时自动刷新右侧实时预览。
- tab 模式下，切换到预览页时自动生成最新预览，保证看到的是刚刚编辑后的结果。

### 页面设置与导航

- 纸张大小补充 A5、A6、B4、Legal 等常规预设，便于不同病历与单据版式切换。
- 纸张方向切换后立即刷新设置区状态，并同步驱动预览宽高变化。
- 左侧改为“物料 / 结构树”双标签视图，让结构树拥有完整可视区域，不再被压缩在物料库底部。
- 页面结构树展示页眉、正文、页脚以及分节/组合内子块层级，并继续下钻到字段级节点，支持直接定位到字段属性配置。
- 结构树补充关键词搜索、分区折叠、块级折叠，降低大型模板的纵向浏览成本。
- 字段节点直接展示规则数量与异常计数，让重复字段、悬空规则引用等问题前置暴露在导航层。
- 结构树补充“仅看异常 / 仅看规则”过滤器，便于在复杂模板里快速切到质控视角或规则视角。

### 页眉与页脚编辑

- 页眉 / 页脚区从“列表型块管理”升级为更接近画布心智的卡片式编辑带，降低维护人员在正文之外处理页脚内容时的心智切换。
- 页脚区补充签名页脚、双签名页脚、打印信息、版权说明等常用预设，支持先套模板化骨架再微调字段和文本。
- 页眉 / 页脚支持直接接收左侧物料库拖拽投放，避免必须通过“添加块”菜单逐个创建。
- 页眉 / 页脚块补上上移、下移、复制、删除等快速整理动作，让页脚编排更接近低代码编辑器的操作节奏。
- 拖拽到页眉 / 页脚时会展示明确的前插 / 后插落点反馈，提升页脚重排和补块的确定性。
- 页眉 / 页脚现有块支持直接拖动重排，并在区尾补充插槽占位文案，避免“能加块不能快速重排”的体验断层。

### 页脚运行态

- 页面设置新增“页脚运行态”分组，用于集中配置页码开关、页码格式、页码对齐、页码底边距以及预览操作者。
- 页码不再由模板维护人员手工拼静态文本，而是复用编辑器内核的 pageNumber frame，根据模板配置在运行时渲染真实页码和总页数。
- 静态文本和段落支持系统变量替换，首批覆盖 `{{打印时间}}` 与 `{{操作者}}`，用于页脚打印追溯、操作人标识等场景。
- 页码格式支持 `{{页码}}` 与 `{{总页数}}`，便于模板维护人员定义“第 X / Y 页”等院内常见格式。

### 指标反馈

- 增加结构块数、字段数、规则数、页眉块数、页脚块数、配置问题数。
- 指标随字段、块、页眉页脚、布局属性变化实时刷新。
- 配置问题数大于 0 时使用风险视觉提示。

### 物料库

- 左侧物料库增加搜索框。
- 基础块按结构布局、内容字段、高级组件分组。
- 医疗片段独立成组，保留患者基本信息、生命体征、签名、护理记录、出院信息等业务片段。
- 每个物料展示图标、名称和说明，提升可理解性。
- 物料库支持直接拖拽到正文画布，也支持拖入分节/组合内部，降低搭建成本。

### 画布

- 空画布提示升级为明确的编辑路径引导。
- 块卡片增加序号，便于讨论和定位。
- 块卡片增加类型色条，区分分节、字段行、段落、表格、静态文本等。
- 非分节块增加摘要信息，减少打开属性面板前的判断成本。
- 画布增加轻量网格背景，强化设计器语境。
- 拖拽投放增加明确的前插 / 后插指示线，并在空画布与嵌套子块容器中显示占位态，降低放置歧义。
- 当正文或嵌套容器为空时，也可以直接把左侧物料拖入空态区域完成首个块投放。
- 画布补充缩放工具条、顶部 / 左侧标尺，支持复杂长文档的版式审阅与尺度感知。
- 迷你地图在真实编辑链路中的收益偏低，回收为更聚焦的画布空间，把导航重心重新收敛到结构树与主画布。

### 属性面板

- 右侧面板宽度提升，输入控件圆角和视觉层级统一。
- 标题从英文式大写改为更贴近业务后台的中文层级。
- 属性栏重构为更接近低代码平台的卡片式分区，弱化工程面板感，提升长时间配置时的可读性与扫描效率。
- 输入控件统一为更明确的聚焦态、边界和阴影层级，减少不同块配置之间的视觉跳变。
- 输入框、数字框和文本域改为失焦后再统一提交，避免输入过程中的闪烁、光标跳动和中文输入法组合态被打断。
- 常用配置尽量提供预设入口，首批为页边距预设卡片，减少维护人员从零调参的成本。
- 对标低代码竞品继续扩展预设化能力，字段属性支持短文本、长文本、日期、数值、是/否选择、签名等快配卡片。
- 文本样式配置支持正文、一级标题、提示说明、风险强调等样式预设，帮助维护人员先套用再微调。
- 行级属性不再在标签下方常显默认值，改为在输入控件中保留默认占位参考；配置帮助说明则收敛到标签旁的即时悬浮 tooltip icon，减少视觉噪音并保持提示可达。
- 段落补齐字体、字号、字高、行距、颜色、高亮、粗斜体、对齐、自动换行等配置。
- 分节标题补齐标题样式配置，支持按模板块定制标题视觉。
- 静态文本补齐字体、高度、行距、颜色、高亮等常见文本属性。
- 修复静态文本输入时因属性栏重渲染导致的丢焦点问题，保证连续录入和光标位置稳定。
- 字段行补齐自动换行、组补齐同行分隔符、分隔线补齐线宽和虚线样式。
- 分隔线进一步补齐颜色、长度、对齐、上下留白等版式项，方便制作页头分区线、签章线和诊疗模块分隔线。
- 分隔线补充预设样式卡片，覆盖实线、虚线、签章线、标题分割线等常用医疗文书场景。
- 表格补齐表头高度、数据行高、边框宽度、边框颜色、列宽等常见版式项。
- 字段补齐默认值、前置文案、后置文案、允许删除、高亮色等常用配置。

- [x] 添加、删除、复制、移动、修改字段后指标会刷新。

## 下一步建议

1. 增加字段绑定面板，支持从 HIS 业务字段中心直接拖入字段。
2. 增加块级模板片段收藏和复用。
3. 增加画布历史时间线，让设计者可以看到本次编辑的结构变化。
4. 为结构树补充保存自定义筛选视图，服务规则设计、质控排查、发布前巡检等场景。
5. 为属性栏补充分组折叠、批量重置和常用配置收藏，继续向成熟低代码工作台靠拢。
